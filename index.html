This is the initial landing page for the COMP122 Instructional Tools.

As development proceeds, each of the following individual web page provides

  • Data Segment: Demonstrates how the .data segment values are positioned and aligned in memory
  • Decimal-to-Binary: Demonstates how to convert a Real Decimal to a Real Binary number
  • Floating-Point: Demonstrates how to encode a Real Binary to a IEEE floating pointing
  • Frame: Demonstrates how values are postioned in a subroutine frame on the stack
  • Others?

Components

This particular webapp will be built around a number of components that are reused. Said components will be build using various vue directives. Current components include:

  • Register Bank: Using the design of the MIPS archicture.
  • Data Segment: A viewable area of memory
  • Source Code: An editable area for including MIPS assemble code.
  • Data Source: An editable area for including MIPS data directives. (A subset of the Source code component in which only the .data segments are active.)
  • Memory Configuration: A control panel to adjust the memory layout.
